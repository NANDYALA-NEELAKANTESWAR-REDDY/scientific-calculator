# Scientific Calculator

A fully functional web-based scientific calculator with a modern, responsive design.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Scientific Functions**: 
  - Trigonometric: sin, cos, tan (in degrees)
  - Logarithmic: log (base 10), ln (natural log)
  - Power: x^y for exponentiation
  - Square root: √
- **Mathematical Constants**: π (pi), e (Euler's number)
- **Memory Functions**: M+, M-, MR (Memory Recall), MC (Memory Clear)
- **Additional Features**:
  - Clear (C) and Clear Entry (CE)
  - Backspace for editing
  - Toggle sign (±)
  - Decimal point support
  - Keyboard support

## Usage

1. Open `index.html` in any modern web browser
2. Use mouse clicks or keyboard to perform calculations

### Keyboard Shortcuts

- **Numbers**: 0-9
- **Operations**: +, -, *, /
- **Decimal**: .
- **Calculate**: Enter or =
- **Clear**: Escape
- **Backspace**: Delete last digit

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and responsive design
- `calculator.js` - Calculator logic and functionality
